# sie-uemoaplateforme
Je souhaite convertir un fichier de système d'information énergétique en plateforme web. Le projet permet de charger mon fichier (Fiche collecte SIE-UEMOA _Final_2020.xlsm) sur la page html, qui lira automatiquement les 30 feuilles et créera les onglets de navigation pour consulter los données de collecte directement dans le navigateur
